# vision
Simple drawing program with vim-like keybinds

## Installation
Requires a C compiler, GNU make, GLEW and GLFW3.
Run (as root):
```
make install
```

## Keybinds
h: scroll left

j: scroll down

k: scroll up

l: scroll right

J: zoom out

K: zoom in

ZZ: exit

ZQ: exit

zz: center and reset zoom
